06. PWC170 / PWCaf / МПК (VO2max)

Назначение:
Функциональная оценка физической работоспособности по реакции ЧСС на две нагрузки.
Для обычного пользователя без контроля применять осторожно.

UI-предупреждение:
"PWC170 и МПК — функциональная оценка. Не проводить тест при противопоказаниях, повышенном давлении, плохом самочувствии, беременности или сердечно-сосудистых жалобах."

Вход:
- sex: female | male
- age: number
- weightKg: number
- w1: number — мощность/нагрузка 1
- w2: number — мощность/нагрузка 2
- hr1: number — ЧСС после нагрузки 1
- hr2: number — ЧСС после нагрузки 2
- mode: "pwc170" | "pwcaf"

Валидация:
age > 0
weightKg > 0
w1 >= 0
w2 > w1 желательно
hr1 > 0
hr2 > 0
hr2 !== hr1

Если hr2 === hr1:
valid: false
warning: "Нельзя рассчитать PWC: HR2 и HR1 равны. Проверьте ЧСС."

Если w2 <= w1:
warning: "Проверьте нагрузки: W2 обычно должна быть выше W1."

Если hr2 <= hr1:
warning: "Проверьте ЧСС: HR2 обычно должна быть выше HR1."

Формулы:

Классический PWC170:
PWC170 = W1 + ((W2 - W1) * (170 - HR1)) / (HR2 - HR1)

Возрастной/адаптированный вариант PWCaf:
indicatorHR = (220 - age) * 0.87
PWCaf = W1 + ((W2 - W1) * (indicatorHR - HR1)) / (HR2 - HR1)

МПК / VO2max:
VO2max = (1.7 * PWC + 1240) / weightKg

Округление:
PWC: 1 знак
VO2max: 1 знак
indicatorHR: 0 знаков
