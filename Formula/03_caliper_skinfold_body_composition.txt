03. КЖС / КАЛИПЕР / СОСТАВ ТЕЛА: % ЖИРА / ЖИРОВАЯ МАССА / LBM

Назначение:
Ориентировочная тренерская оценка состава тела по кожно-жировым складкам.
Точность зависит от правильности замеров.
Не является медицинской диагностикой.

Вход:
Общие:
- sex: female | male
- age: number
- heightCm: number
- weightKg: number

Складки:
- forearm — предплечье
- armFront — плечо спереди
- armBack — плечо сзади
- scapula — лопатка
- abdomen — живот
- thigh — бедро
- calf — голень/икра

Для мужчин дополнительно:
- chest — грудь

Валидация:
heightCm > 0
weightKg > 0
age > 0
Все складки >= 0
Для male chest обязателен и >= 0

Если общая ошибка:
valid: false
warning: "Проверьте вводные: рост/вес должны быть больше 0, складки не могут быть отрицательными."

Если мужчина без chest:
valid: false
warning: "Для мужского расчёта добавьте складку груди."

ФОРМУЛЫ

1. Площадь поверхности тела / BSA
bodySurfaceArea = sqrt((heightCm * weightKg) / 3600)
Округлить до 3 знаков.

2. Сумма складок

Женщины:
sumFolds = forearm + armFront + armBack + scapula + abdomen + thigh + calf
averageSkinfold = sumFolds / 14

Мужчины:
sumFolds = forearm + armFront + armBack + chest + scapula + abdomen + thigh + calf
averageSkinfold = sumFolds / 16

Почему деление на 14/16:
Методика тренерского протокола, где используется усреднение складок по двум сторонам/коэффициентам протокола.

3. Жировая масса
fatMassKg = averageSkinfold * bodySurfaceArea * 1.3

4. Процент жира
fatPercent = fatMassKg / weightKg * 100

5. Безжировая масса / LBM
lbmKg = weightKg - fatMassKg

Округление:
- sumFolds: 1 знак
- averageSkinfold: 2 знака
- bodySurfaceArea: 3 знака
- fatMassKg: 1 знак
- fatPercent: 1 знак
- lbmKg: 1 знак

ИНТЕРПРЕТАЦИЯ % ЖИРА

Женщины:
18–29:
- fatPercent < 20: ниже ориентировочного диапазона
- 20 <= fatPercent <= 28: ориентировочный диапазон
- 28 < fatPercent < 29: выше нормы / погранично
- fatPercent >= 29: повышен

30–39:
- fatPercent < 22: ниже ориентировочного диапазона
- 22 <= fatPercent <= 30: ориентировочный диапазон
- fatPercent >= 30: повышен

40–49:

- 24 <= fatPercent <= 32: ориентировочный диапазон
- fatPercent >= 32: повышен

- fatPercent >= 19: повышен
