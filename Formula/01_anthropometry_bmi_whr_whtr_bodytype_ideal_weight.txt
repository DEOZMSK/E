01. АНТРОПОМЕТРИЯ: BMI / WHR / WHtR / ТИП ТЕЛОСЛОЖЕНИЯ / ОРИЕНТИРОВОЧНЫЙ ВЕС

Назначение:
Базовый тренерский блок для оценки стартовых данных клиента.
Не является медицинской диагностикой.

Вход:
- sex: female | male
- age: number
- heightCm: number
- weightKg: number
- waistCm: number
- hipCm: number
- wristCm: number

Валидация:
heightCm > 0
weightKg > 0
waistCm > 0
hipCm > 0
wristCm > 0
age > 0 желательно, но в формулах BMI/WHR/WHtR возраст напрямую не нужен.

Если ошибка:
valid: false
warning: "Проверьте вводные: рост, вес, талия, бёдра и запястье должны быть больше 0."

ФОРМУЛЫ

1. BMI / ИМТ
heightM = heightCm / 100
BMI = weightKg / (heightM * heightM)
Округлить до 2 знаков.

Комментарии BMI:
- BMI < 18.5:
  "По введённым данным BMI ниже ориентировочного диапазона."
- 18.5 <= BMI < 25:
  "По введённым данным BMI в ориентировочном диапазоне."
- 25 <= BMI < 30:
  "По введённым данным BMI выше ориентировочного диапазона."
- BMI >= 30:
  "По введённым данным BMI значительно выше ориентировочного диапазона, нужна осторожность в прогрессии нагрузки."

Важно:
BMI не использовать как единственный вывод о форме. У тренированных людей с высокой мышечной массой BMI может искажать картину.

2. WHR — waist-to-hip ratio / талия-бёдра
WHR = waistCm / hipCm
Округлить до 2 знаков.

Пороги WHR:
Мужчины:
- WHR > 0.90:
  "По введённым данным WHR указывает на абдоминальный тип, повышенное внимание."
- 0.88 <= WHR <= 0.90:
  "По введённым данным WHR — пограничное значение."
- WHR < 0.88:
  "По введённым данным WHR без повышенного внимания."

Женщины:
- WHR > 0.85:
  "По введённым данным WHR указывает на абдоминальный тип, повышенное внимание."
- 0.83 <= WHR <= 0.85:
  "По введённым данным WHR — пограничное значение."
- WHR < 0.83:
  "По введённым данным WHR без повышенного внимания."

3. WHtR — waist-to-height ratio / талия-рост
WHtR = waistCm / heightCm
Округлить до 2 знаков.

Пороги WHtR:
- WHtR < 0.40:
  "По введённым данным WHtR ниже стандартного диапазона."
- 0.40 <= WHtR < 0.50:
  "По введённым данным WHtR без повышенного риска."
- 0.50 <= WHtR < 0.60:
  "По введённым данным WHtR: повышенное внимание."
- WHtR >= 0.60:
  "По введённым данным WHtR высокий риск, нужна осторожность."

4. Тип телосложения по запястью

Мужчины:
- wristCm < 17: астеник
- 17 <= wristCm <= 20: нормостеник
- wristCm > 20: гиперстеник

Женщины:
- wristCm < 15: астеник
- 15 <= wristCm <= 18: нормостеник
- wristCm > 18: гиперстеник

5. Ориентировочный вес по типу телосложения

Мужчины:
- астеник: estimatedWeightKg = heightCm * 0.375
- нормостеник: estimatedWeightKg = heightCm * 0.39
- гиперстеник: estimatedWeightKg = heightCm * 0.41

Женщины:
- астеник: estimatedWeightKg = heightCm * 0.325
- нормостеник: estimatedWeightKg = heightCm * 0.34
- гиперстеник: estimatedWeightKg = heightCm * 0.355

Округлить estimatedWeightKg до 1 знака.

В UI не писать “идеальный вес” как обязательную цель. Лучше:
- “ориентировочный вес”
- “расчётный ориентир по типу телосложения”

Рекомендуемый output:
{
  valid: true,
  bmi,
  whr,
  whtr,
  bodyType,
  estimatedWeightKg,
  bmiComment,
  whrComment,
  whtrComment
}
