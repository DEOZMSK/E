05. ГИБКОСТЬ: НАКЛОН ВПЕРЁД / SIT-AND-REACH

Назначение:
Ориентировочная оценка гибкости / подвижности задней поверхности тела.
Не выполнять через боль.
Осторожно при травмах спины, беременности, выраженном дискомфорте.

Вход:
- age: number
- reachCm: number

Валидация:
age > 0
reachCm должен быть числом
Если возраст вне шкалы:
valid: false
warning: "Для этого возраста нет шкалы в таблице."

Нормы:
- age < 25: норма > 36 см
- 25 <= age <= 30: норма 33–35 см
- 30 < age <= 45: норма 29–33 см
- 45 < age <= 60: норма 21–29 см

Интерпретация:
Для age < 25:
- reachCm < 36: ниже ориентира
- reachCm >= 36: в ориентире / выше ориентира можно считать хорошим результатом

Для 25–30:
- reachCm < 33: ниже ориентира
- 33 <= reachCm <= 35: в ориентире
- reachCm > 35: выше ориентира

Для 31–45:
- reachCm < 29: ниже ориентира
- 29 <= reachCm <= 33: в ориентире
- reachCm > 33: выше ориентира

Для 46–60:
- reachCm < 21: ниже ориентира
- 21 <= reachCm <= 29: в ориентире
- reachCm > 29: выше ориентира

Комментарии:
Если ниже ориентира:
"Есть смысл добавить мягкую мобилизацию и растяжку без боли."

Если в ориентире:
"Гибкость в ориентировочном диапазоне."

Если выше ориентира:
"Хорошая гибкость по введённым данным."

Output:
{
  valid: true,
  reachCm,
  status: "ниже ориентира" | "в ориентире" | "выше ориентира",
  comment
}
