11. LEGACY / EXCEL NOTES

Этот файл описывает старые Excel-логики, которые могут встретиться в исходных таблицах.
Не использовать их в пользовательском отчёте без явного решения.

====================================================================
СТАРАЯ ФОРМУЛА BMR ЧЕРЕЗ % ЖИРА
====================================================================

BMR = sexCoefficient * weightKg * 24 * fatIndex + calorieAdjustment

sexCoefficient:
- male: 1.0
- female: 0.9

fatIndex для мужчин:
- 10–14%: 1.0
- >14–20%: 0.95
- >20–28%: 0.90
- >28%: 0.85

fatIndex для женщин:
- 14–18%: 1.0
- >18–28%: 0.95
- >28–38%: 0.90
- >38%: 0.85

TDC = BMR * activityCoefficient

Старая схема БЖУ:
- белки 20% калорий
- жиры 30% калорий
- углеводы остаток
- белки и углеводы: 4 ккал/г
- жиры: 9 ккал/г

Решение для текущего проекта:
Основной пользовательский расчёт делать через Mifflin-St Jeor + БЖУ по граммам/кг.
Старую схему хранить как legacy, если нужна сверка Excel.

====================================================================
ФОРМУЛЫ ИДЕАЛЬНОГО ВЕСА
====================================================================

В исходниках встречаются разные формулы:

1. Брока-Бругша:
- рост 150–165: M = L - 100
- рост 165–175: M = L - 105
- рост 175–185: M = L - 110

2. Лоренса:
M = L - (100 - ((L - 150) / 4))

3. Формула с учётом пола, роста, возраста:
Мужчины:
M = 50 + 0.75 * (L - 150) + 0.25 * (W - 21)

Женщины:
M = 50 + 0.34 * (L - 150) + 0.2 * (W - 21)

4. По типу телосложения:
Мужчины:
- астеник: 0.375 * L
- нормостеник: 0.39 * L
- гиперстеник: 0.41 * L

Женщины:
- астеник: 0.325 * L
- нормостеник: 0.34 * L
- гиперстеник: 0.355 * L

Решение для текущего проекта:
В /trainers использовать формулу по типу телосложения как основной ориентир.
Не писать “идеальный вес”, писать “ориентировочный вес”.
